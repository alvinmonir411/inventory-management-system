# Dealer ERP Monorepo

This repository is organized as a simple monorepo for a web-based Dealer ERP system for Bangladesh distribution businesses.

## Folders

- `backend/` - NestJS backend application, database migrations, and server-side business logic.
- `frontend/` - Next.js frontend application for the web interface.
- `docs/` - Project documentation, architecture notes, and business/workflow references.

## Notes

- Frontend and backend are kept fully separate.
- Business modules are intentionally not initialized yet.
- The structure is minimal and ready for production-oriented development.
